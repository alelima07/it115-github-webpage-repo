IT115 Final Project

Link to Project Doc Info about project: This web page is part of an assignment for our Intro to Software Development class at Seattle Central Community College. A group of 9 students had the opportunity to collaborate together to create two Github Repos, a web page, a web application and three google documents that would take you through a step by step tutorial on how to set up a MEAN stack on a google cloud instance and then demonstrate that stack on a web app /clon it.

Link to Stack Tutorial How to build a MEAN stack? This is a link to the MEAN stack tutorial document that was created for this project. This tutorial will take you through a step by step process on how to set up a MEAN stack on a Google Cloud instance. The MEAN stack is comprised of 4 parts: MongoDB, Express, AngularJS and Node.js. If you would like to view the tutorial, please click on the link above.

Link to WebApp Tutorial How to clone a web app repo? This is a link to the Web application tutorial that was created for this project. If you'd like to see a demonstration of the MEAN stack on a web app, please click above and utilize our step by step process. This tutorial will provide all the steps needed to demonstrate your stack, clearly outlining each step.

Link to WebApp-Repo Link to WebPage-Repo Github repositories: Our two repositories contain all of the project files, including link to website, link to MEAN stack tutorial, and link to web app tutorial. You can also see what roles there are and who take care of them. There are a couple of files you can see on the master branch that the collaborators share. Thank you!
